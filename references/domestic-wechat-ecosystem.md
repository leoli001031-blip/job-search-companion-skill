# 微信生态求职沟通

用于国内求职中通过微信发送简历、跟进面试、求内推或维护个人职业形象的场景。

## 使用原则

- 微信不是正式求职信渠道，核心是短、清楚、方便对方转发。
- 不要一次发送大段自我介绍。优先发送：一句话定位 + PDF 简历 + 可选作品链接。
- 如果对方用手机查看概率高，准备 PDF 附件；作品型岗位可另备 1 页长图或截图集。
- 文件名写清楚：`姓名-岗位方向-城市.pdf`。如用户愿意，也可加电话后四位便于 HR 检索；不要默认暴露完整手机号。
- 重要信息不要只发语音；涉及时间、地点、薪资、Offer 等内容尽量留文字记录。

## 加 HR 微信后的自我介绍

```text
您好，我是 [姓名]，刚才在 [平台] 上沟通过 [岗位名称]。我有 [相关经历/项目]，和岗位里的 [核心要求] 比较匹配。简历发您，方便时请您查看。
```

## 微信发简历

推荐顺序：

1. 先发一句说明。
2. 发送 PDF 简历。
3. 如岗位需要作品，再补作品集链接或 1-3 张关键截图。

模板：

```text
您好，这是我的简历 PDF。我的匹配点主要是：[相关经历 1] + [可证明项目/作品] + [岗位核心能力]。作品链接我也附在简历里，如需我可以再发项目截图或演示视频。
```

## 面试后微信跟进

适用：面试结束后 24 小时内，确实有必要补充材料或表达感谢。

```text
您好，感谢今天的沟通。我对岗位里的 [核心任务/业务场景] 更清楚了，也补了一份 [项目截图/案例说明/作品样例]，供您和业务负责人参考。辛苦。
```

不要写：

- “我真的非常非常想加入贵公司。”
- “什么时候给结果？”
- 长篇复盘面试过程。

## 微信群 / 熟人转发内推

准备一段 60-100 字的转发文案：

```text
朋友们好，我最近在看 [目标岗位/方向] 相关机会。背景是 [相关经历 1] + [相关经历/项目 2]，目标城市 [城市/远程/可异地] 可考虑。简历可私发，感谢帮忙留意。
```

## 朋友圈职业形象

适合设计、内容、运营、产品、AI 应用、咨询类岗位。

可以保留：

- 作品截图、项目复盘、学习笔记、活动现场记录。
- 对行业和工具的观察。
- 不泄露客户信息的案例片段。

谨慎处理：

- 情绪化吐槽前公司/客户/同事。
- 过多炫耀式内容。
- 涉及隐私、政治、攻击性表达的内容。

## 微信沟通边界

- 不在微信里发送身份证、银行卡、薪资流水等敏感材料，除非确认对方身份、用途和必要性。
- 对方要求补充敏感信息时，先问清楚用途、接收主体和保存方式。
- 面试时间、地点、薪资口径、入职材料清单尽量留文字。
