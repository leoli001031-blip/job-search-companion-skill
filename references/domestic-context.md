# 国内求职语境

用于国内求职、校招、实习或中文简历时的调整。不要机械套用不符合中文招聘习惯的简历规范。

## 常见阶段

```text
实习
校招 / 应届
社招
转岗 / 转行
管培生
考公/事业单位/国企类申请
自由职业/作品型求职
```

不同阶段关注点不同：

- 实习/校招：教育、实习、校园经历、项目、竞赛、证书、潜力。
- 社招：岗位匹配、可迁移成果、业务结果、稳定性、过往职责深度。
- 管培生：学习能力、组织协调、表达、领导潜力、校园/实习证明。
- 作品型岗位：作品质量、审美/表达/传播、案例过程、客户或用户反馈。

## 国内材料常见来源

- 招聘平台资料：BOSS 直聘、猎聘、智联招聘、前程无忧、牛客、实习僧、应届生求职网等。
- 内容/作品平台：公众号、小红书、B 站、视频号、知乎、站酷、Behance、个人网站等。
- 学校材料：成绩单、奖学金、竞赛证书、论文/毕设、课程项目、学生干部/社团证明。

平台名单只作为常见例子，不作为实时完整清单。

## 国内执行链路

国内求职常见链路不只是“投简历 -> 面试”：

```text
平台资料完善
在线简历 / 附件简历 / 作品集
主动沟通或网申
测评 / 笔试 / 群面
业务面 / 主管面 / HR 面 / 终面
谈薪 / 背调 / 书面 offer
应届生三方 / 网签 / 去向登记
入职材料
```

如果任务涉及国内求职执行，优先读取：

```text
references/domestic-delivery-channels.md
references/domestic-interview-rounds.md
references/group-interview-campus.md
references/tracking.md
```

## 求职阶段与周期策略

```markdown
| 周期 | 策略 |
| --- | --- |
| 急投，1-2 周 | 先做最小可投版本：简历、目标 JD、打招呼、风险口径；不追求全量作品集 |
| 正常找，1-3 月 | 边投边补：在线简历、附件简历、作品/案例页、面试故事库同步迭代 |
| 长期准备，3-6 月 | 先补项目、证书、作品集和目标岗位能力地图，再集中投递 |
```

## 0 经验 / 材料少候选人的应急路径

- 把课程项目、课堂作业、社团活动、志愿服务、兼职整理成项目或经历。
- 从日常场景提取可迁移能力：沟通、执行、数据整理、服务意识、活动组织。
- 用 1-2 周做 1-2 个可展示的小作品或案例复盘。
- 先投匹配度最高的少量岗位，从反馈中迭代，不要一开始海投。

## HR 快速扫简历逻辑

国内 HR 初筛通常先看：

```text
岗位关键词是否匹配
学校/专业/年限/证书是否过硬门槛
时间线是否稳定
项目或工作结果是否可信
联系方式和附件是否规范
```

首屏要让人快速看出“投什么岗位、有什么证据、有没有明显硬伤”。

## 边界说明

本 Skill 主要覆盖市场化求职。考公、考编、军队文职、部分国企央企专项招聘涉及申论、行测、政审、体检、编制、档案和组织关系等专门流程，需要另查针对性资料。

## 中文简历倾向

- 表达更直接、紧凑，少用夸张形容词。
- 国内 HR 常快速扫：学校/专业/时间线/岗位关键词/实习公司/项目结果/证书。
- 学生简历可以把教育背景放前面；社招通常工作经历更靠前。
- 如果 JD 明确卡学历/专业/证书，教育背景应靠前；如果候选人是转型或项目驱动型，但学历能覆盖门槛，也建议在首屏放出教育背景，减少 HR 初筛疑虑。
- 对传统行业、商管、国企/央企、校招/实习、管培等场景，简历结构要更像中文招聘平台：求职意向、教育背景、个人优势、专业技能、经历/项目；不要完全采用作品集式排序。
- 证书、获奖、实习、学生干部、竞赛可以有价值，但必须和岗位相关或能证明潜力。
- 不要为了“丰富”堆无关奖项；每一项都要能回答“它证明了什么”。
- 技能区要按 JD 能力簇分组，而不是无序罗列。推荐顺序：岗位核心能力、工具/方法、业务场景、沟通协作、语言/证书。

## 中文 Bullet 写法

优先：

```text
在什么场景下，负责什么，做了什么动作，交付了什么，带来什么结果/证据。
```

例子：

```text
负责学院迎新活动志愿者排班与现场协调，统筹 30 名志愿者完成 800+ 名新生报到引导，沉淀现场问题清单与第二年流程优化建议。
```

比下面更好：

```text
参与迎新活动，认真负责，沟通能力强。
```

## 敏感信息处理

- 照片：不是所有岗位都需要。除非用户希望或岗位明确要求，否则先做无照片版本。
- 政治面貌、籍贯、民族、婚育等信息：不要默认加入；只有明确需要时处理。
- 薪资、离职原因、空窗期：先作为内部面试准备，不默认写进简历。
- 被问到与岗位无关的婚育、家庭、性别、地域、年龄等问题时，优先转回岗位可用性和投入条件；不要主动透露无关隐私。
