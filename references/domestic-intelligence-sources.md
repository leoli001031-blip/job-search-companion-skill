# 国内公司情报收集

用于在官方信息之外，收集面试、薪资、团队氛围和岗位真实情况。所有非官方信息都要交叉验证，不要把匿名爆料当事实。

## 信息源分层

强事实：

- 公司官网、招聘 JD、工商信息、上市公司公告、政府/学校/媒体公开报道。
- 国家企业信用信息公示系统、天眼查、企查查等工商和风险信息源，用于查主体、经营异常、法律诉讼、股权结构等线索。

中等参考：

- 牛客面经、看准网面试经验、职友集薪资范围、猎聘薪资报告、脉脉公开讨论。

弱参考：

- 小红书吐槽、匿名爆料、离职博主、单篇情绪化帖子。

## 脉脉

适合查：

- 公司氛围、部门变动、裁员传闻、薪资爆料、内部员工视角。

使用方法：

- 搜公司名 + 岗位名 + 城市。
- 按时间排序，优先看近 6-12 个月。
- 同一结论至少找 2-3 个来源交叉验证。
- 对明显情绪化内容做降权处理。
- 匿名区可信度低于实名内容；匿名爆料只能作为风险线索，不能当事实。

面试中使用：

```text
我看到公开信息里公司在推进数字化和 AI 相关方向，也想进一步了解这个岗位目前更偏从 0 调研，还是已有明确试点场景？
```

不要说：

```text
我看脉脉上说你们部门很乱。
```

## 牛客

适合查：

- 校招/互联网岗位面经、笔试题、HR 面问题、薪资开奖。

使用方法：

- 公司名 + 岗位名 + 面经。
- 按时间筛选，旧面经只作题型参考。
- 总结高频问题，而不是背答案。

## 小红书 / 知乎

适合查：

- 公司体验、行业体验、求职避坑、岗位日常。

风险：

- 幸存者偏差强。
- 离职/踩坑内容情绪浓度高。
- 不能直接当作面试结论。

## 看准网 / 职友集

适合查：

- 薪资区间、面试难度、公司评价。

处理方式：

- 只作为区间参考。
- 和招聘 JD 薪资、猎头反馈、同岗位 Offer 信息对比。

## 30 分钟情报预算

```text
10 分钟：官网 / JD / 公开报道
8 分钟：招聘平台和公司页
7 分钟：脉脉 / 牛客 / 看准
5 分钟：整理面试可问问题
```

## 输出模板

```markdown
| 信息 | 来源 | 可信度 | 对求职动作的影响 |
| --- | --- | --- | --- |
|  | 官方/半官方/匿名 | 高/中/低 | 简历/面试/谈薪/风险 |
```

## 来源与更新线索

- 脉脉官网定位为职场社区和招聘平台：https://maimai.cn/
- 牛客网定位为互联网求职学习平台，包含刷题、面经和招聘：https://www.nowcoder.com/
- 国家企业信用信息公示系统：https://www.gsxt.gov.cn/
- 非官方平台信息只作为线索，不作为未经验证的事实。
