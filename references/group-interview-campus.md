# 校招、实习与群面

用于校招、实习、管培生、无领导小组讨论、案例分析、网申、测评、笔试、三方协议。

## 校招 / 实习流程

常见流程可能包括：

```text
信息收集 -> 网申/在线简历 -> 测评/笔试 -> 群面/无领导 -> 业务面/专业面 -> HR 面 -> 终面 -> offer call -> 三方/网签 -> 入职材料
```

不同公司可能跳过部分环节。不要用一个公司流程推断所有公司。

## 校招时间窗口

每年会调整，以公司当年公告和学校就业网为准。常见节奏：

```text
秋招提前批：7-8 月
秋招正式批：9-11 月
秋招补录：11-12 月
春招：3-5 月
春招补录：5-6 月
暑期实习招聘：3-5 月
```

策略：

- 提前批适合准备充分、目标清楚的候选人。
- 正式批岗位多但竞争集中。
- 补录岗位少，节奏快，要准备压缩版材料。
- 暑期实习常是转正和秋招前置入口。

## 校招信息源

- 学校就业信息网和学院通知。
- 企业校招官网和校招公众号。
- 应届生求职网、牛客校招版、BOSS/智联/实习僧校招专区。
- 班级群、院系群、校友群。
- 线下宣讲会、双选会和企业开放日。

## 网申与开放题

网申不是机械填表，要保持和简历一致：

```text
教育经历、项目经历、实习经历、获奖证书、时间线必须一致
开放题要贴公司/岗位，不写泛泛鸡汤
同一个素材可以复用，但要按岗位改关键词和重点
投递后保存 JD、开放题答案和投递时间
```

开放题常见方向：

```text
为什么选择公司/岗位
最有成就感的经历
一次失败或困难
团队协作/冲突
职业规划
对行业/产品/业务的理解
```

## 测评 / 笔试

```markdown
| 类型 | 准备重点 | 记录 |
| --- | --- | --- |
| 性格测评 | 前后一致，贴近真实工作风格 | 不伪装极端人格 |
| 行测/能力测评 | 时间管理、题型熟悉 | 记录题型和薄弱项 |
| 专业笔试 | 岗位基础知识、案例题 | 保留复盘 |
| 作品/视频/试讲 | 结构清楚、证据充分 | 保留最终版和反馈 |
```

刷题与练习资源：

- 行测/综合能力：粉笔、中公、华图等题库。
- 编程/技术：牛客、LeetCode、目标公司往年题。
- 面经：牛客、校友群、企业校招论坛。
- 性格测评：保持前后一致，不要伪装极端人格。

## 无领导小组讨论

核心不是抢 leader，而是贡献有效推进。

面试官常看：

```text
有效表达
倾听和整合
逻辑框架
时间观念
协作和情绪稳定
推进分歧
总结能力
```

## 群面角色策略

```markdown
| 角色 | 适合人群 | 做什么 | 风险 |
| --- | --- | --- | --- |
| 破题者 | 逻辑强、敢开场 | 定义问题、提出框架 | 不要开场太武断 |
| 框架推进者 | 结构化强 | 整合观点、推进讨论 | 不要只写框架不贡献内容 |
| 时间提醒者 | 稳定细心 | 控制进度、提醒阶段 | 不要频繁打断 |
| 信息整合者 | 倾听强 | 总结共识和分歧 | 不要变成复读 |
| 总结陈词者 | 表达稳 | 代表小组输出结果 | 不要抢总结且内容不熟 |
```

## 群面流程模板

```text
1. 快速读题，标出目标、限制、交付物。
2. 用 20-40 秒提出问题理解或讨论框架。
3. 听取他人观点，补充证据或分类。
4. 中段推动取舍：我们现在需要在 A/B 之间做选择。
5. 时间不足时提醒：还剩 X 分钟，建议先定结论再补理由。
6. 结尾总结：结论 -> 3 个理由 -> 风险和补充。
```

## 群面避坑

```text
不要全程沉默
不要为了当 leader 抢话
不要攻击队友
不要只重复别人观点
不要陷入细枝末节
不要不顾时间一直展开
不要总结自己没有参与理解的内容
```

## 三方协议 / 网签提醒

应届生拿到 offer 后，确认：

```text
是否需要签三方/网签
签约主体是否和 offer 公司一致
岗位、工作地、薪资、违约条款是否清楚
档案、户口、报到证/去向登记要求
学校审核流程和时间
毁约成本和后续影响
```

三方、劳动合同和正式入职条款不要混为一谈；关键承诺尽量写入书面材料。

违约金提醒：

- 三方违约金、毁约流程和改派规则由学校、公司和协议共同决定。
- 签三方前确认违约金金额、支付条件、解约函出具时间。
- 多 Offer 未决时，不要过早签约；如果必须签，保留沟通记录。
- 毁约前先确认新 Offer 是否为书面 Offer，避免两边落空。

## 管培生项目差异

管培生不等于普通校招岗位，常见特点：

- 轮岗周期和轮岗部门。
- 阶段性答辩或淘汰机制。
- 定岗规则和城市分配。
- 薪资、培养资源和晋升预期可能写得很漂亮，但要问清落地方式。

需要追问：

```text
轮岗多久？最后如何定岗？是否有淘汰？导师机制如何？定岗城市是否可选？
```
