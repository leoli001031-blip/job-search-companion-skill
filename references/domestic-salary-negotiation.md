# 国内谈薪深度模块

用于国内 Offer 沟通、谈薪、薪资结构拆解、五险一金和背调前薪资证明问题。

## 必问清单

拿到口头 Offer 或进入 HR 谈薪时，至少确认：

- 月薪是税前还是税后。
- 年薪口径如何计算：12 薪、13 薪、14 薪、绩效奖金、年终奖是否固定。
- 试用期多久，试用期工资是否打折。
- 五险一金缴纳城市、基数和比例。
- 公积金是否按实际工资缴纳，比例是多少。
- 薪资结构：基本工资、绩效、补贴、餐补、交通补贴、加班费。
- 绩效部分占比、发放周期和考核规则。
- 是否有竞业限制、服务期、培训协议。
- 背调是否要求提供上家薪资流水或离职证明。
- 劳动合同签约主体、发薪主体、社保主体和实际工作地点是否一致。

## 薪资结构拆解

先把“总包”拆成固定、浮动和一次性收入：

```text
年度总包 = 税前月薪 x 固定发薪月数 + 绩效/年终奖 + 补贴 + 股票/期权 + 签字费/安家费
```

其中：

- 固定部分：月薪、固定 13/14 薪、固定补贴。
- 浮动部分：绩效、年终奖、提成、项目奖金。
- 长期激励：股票、期权、虚拟股，需确认归属节奏和离职处理。
- 一次性收入：签字费、搬家补贴、安家费，需确认是否有服务期或返还条款。

Offer 对比表：

```markdown
| 公司 | 岗位 | 税前月薪 | 固定薪数 | 浮动奖金 | 补贴 | 股票/期权 | 社保/公积金基数 | 试用期 | 到手预估 | 风险 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
```

## 服务 / 销售 / 门店岗位薪资拆解

如果 JD 写“综合薪资”“上不封顶”“底薪 + 提成”“多劳多得”“包吃住”“奖金丰厚”，必须拆到可计算：

```markdown
| 项目 | 需要确认 |
| --- | --- |
| 无责底薪 | 是否无责，金额多少，试用期是否打折 |
| 有责底薪 | 达不到指标是否扣底薪，考核周期是什么 |
| 提成规则 | 按销售额、毛利、回款、件数还是客单计算 |
| 阶梯门槛 | 达到多少才有提成，是否清零或跨月累计 |
| 到手范围 | 普通员工真实中位数，不只听“优秀员工最高” |
| 排班工时 | 月休几天、每天几小时、加班和调休怎么处理 |
| 罚款扣款 | 迟到、损耗、差评、退货、客诉是否扣钱 |
| 社保公积金 | 是否缴纳、基数、城市、试用期是否缴纳 |
| 培训/服装/押金 | 是否收费或要求押金，离职是否退还 |
| 签约主体 | 直营、加盟、外包、劳务派遣还是第三方发薪 |
```

不要只用“综合薪资”判断岗位好坏。先算固定收入、可控浮动、工时强度和权益风险。

## 14 薪 / N 薪检查

不要只问“是不是 14 薪”，要拆：

```text
请问 14 薪里额外 2 薪是写入劳动合同的固定发放，还是根据公司经营和个人绩效浮动？通常发放时间是什么时候？
```

风险口径：

- “综合年薪 20-30 万”可能包含浮动绩效。
- “最高可达 14 薪”不等于固定 14 薪。
- “年终奖看公司情况”不能当成确定收入。

## 五险一金

关键不是“有没有”，而是“按什么基数缴”。

```text
请问五险一金是按实际工资作为基数缴纳，还是按当地最低/固定基数缴纳？公积金单位和个人比例分别是多少？
```

五险一金可单独列：

```markdown
| 公司 | 税前月薪 | 年薪口径 | 固定/浮动 | 社保基数 | 公积金基数 | 公积金比例 | 试用期折扣 | 备注 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
```

## 到手收入估算提醒

年度总包不等于到手收入。估算时至少考虑：

- 五险一金个人扣款。
- 个税。
- 公积金个人和公司部分的双边入账。
- 绩效/年终是否固定。
- 补贴是否税前发放。

不同城市、缴纳基数和公积金比例会显著影响到手收入。不要只比较“税前年包”，要比较固定现金流、浮动比例和社保公积金质量。

## 薪资流水 / 证明

原则：

- 不主动提供，除非进入书面 Offer、背调或公司正式核验阶段，并确认用途、接收主体、保存方式和是否需要本人授权。
- 如果要求提供，优先问是否接受脱敏版本；脱敏可遮盖银行卡号、无关流水、完整身份证号和非必要个人信息。
- 如果流程还停留在初筛、初面或普通谈薪阶段，不建议为了“证明诚意”提前发送薪资流水。
- 薪资谈判应基于目标岗位价值、市场区间和个人匹配度，不应只由上一份薪资决定。

回应模板：

```text
我理解公司需要做薪资和背调核验。薪资流水涉及个人隐私，我可以在 Offer 流程明确后按公司要求提供必要材料，也希望薪资沟通主要基于岗位职责、市场区间和我的匹配度来评估。
```

## 股票 / 期权检查清单

如果 Offer 包含股票、期权、虚拟股或长期激励，确认：

- 类型：RSU、期权、虚拟股还是项目分红。
- 总量和归属节奏：几年归属，每年/每季度多少。
- 行权价和当前估值。
- 离职时未归属部分如何处理。
- 税务和行权成本。
- 是否写入正式 Offer 或协议。

纸面价值不等于到手现金。不能把未归属、不可交易或行权成本高的部分直接当工资。

## 试用期风险管理

需要确认：

- 试用期时长。
- 试用期工资是否打折。
- 转正标准是否书面明确。
- 试用期是否缴纳社保公积金。

风险判断：

```markdown
| 情况 | 风险 | 候选人动作 |
| --- | --- | --- |
| 试用期工资低于约定工资 80% | 高 | 要求 HR 明确合同口径 |
| 只说“看表现转正”没有标准 | 中高 | 追问转正考核指标 |
| 试用期不缴社保 | 高 | 谨慎接受，必要时咨询当地规定 |
| 试用期被以“不符合录用条件”辞退 | 中高 | 保存录用条件、考核记录和沟通证据 |
```

## 薪资谈判锚定策略

- 尽量让对方先说明预算区间。
- 给区间，不只给单个底线数字。
- 用“整体 package”沟通，不只盯月薪。
- 当前薪资偏低时，强调岗位价值、市场区间和能力匹配，不让上份薪资完全锚定新 Offer。
- 有多个机会时，只说“我也在推进其他机会”，不要编造虚假 Offer。

## HR 常见压价话术与回应

```markdown
| HR 话术 | 回应方向 |
| --- | --- |
| 你上一份工资多少？ | 可以说明大致区间，但强调这次希望按岗位职责和市场水平评估 |
| 我们预算有限 | 询问固定薪、绩效、年终、五险一金和成长空间是否能整体平衡 |
| 你最低能接受多少？ | 给期望区间下限，同时说明前提是薪资结构清楚 |
| 其他候选人要求更低 | 回到自身匹配点和可交付价值，不参与低价竞赛 |
| 先入职，后面再调 | 询问调薪周期、触发条件和是否能写入书面约定 |
```

## 试用期工资

《劳动合同法》规定试用期工资有下限。实际沟通中可这样问：

```text
请问试用期工资和转正后是否一致？如果有折扣，折扣比例和试用期时长分别是多少？
```

## 谈薪节奏

- 初面：不主动报底线，可给区间或“结合岗位职责沟通”。
- HR 面：确认薪资结构和期望区间。
- 口头 Offer：逐项问清楚固定和浮动。
- 书面 Offer：核对岗位、薪资、试用期、工作地点、入职日期。
