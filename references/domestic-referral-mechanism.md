# 国内内推机制与关系型求职

用于大厂、互联网、传统企业、校友/同事/朋友内推场景。

## 内推不是一句“帮我投一下”

推荐人通常需要：

- 岗位链接或岗位编号。
- 候选人简历。
- 一句话匹配理由。
- 候选人联系方式。

准备内推包：

```text
1. PDF 简历
2. 目标岗位链接 / JD 截图
3. 一句话匹配理由
4. 作品集/项目链接（如需要）
5. 可联系时间
```

## 找谁内推

优先级：

1. 目标部门熟人。
2. 同公司其他部门熟人。
3. 校友 / 前同事 / 朋友的朋友。
4. 脉脉 / 牛客 / 微信群里的公开内推人。
5. 猎头。

## 熟人内推请求

```text
最近我在看贵公司的 [目标岗位/方向]，看到这个岗位和我过往 [相关经历/项目] 比较匹配。方便的话，想请你帮我看下能否内推。我把岗位链接、简历和一句话匹配理由整理好了，不会占用你太多时间。
```

## 陌生人内推请求

```text
您好，打扰了。我在看贵司这个岗位：[岗位链接]。我的背景是 [相关经历 1] + [相关项目/能力 2]，和岗位里的 [JD 核心要求] 比较匹配。请问方便帮忙内推吗？我可以直接发简历和 3 句话匹配理由，感谢。
```

## 一句话匹配理由

```text
[相关行业/业务经验] + [可证明项目/成果]，能做 [岗位核心任务 1]、[岗位核心任务 2] 和 [岗位核心任务 3]。
```

## 跟进节奏

- 第一次发出后 2-3 天无回复，可轻跟一次。
- 推荐人已提交后，不要每天追问。
- 如果拿到面试，及时告知推荐人进展。
- 如果入职成功，再表达感谢；是否有内推奖金由公司制度决定，不主动提分成。

## 内推激励与边界

部分公司会给推荐人内推奖励，金额、发放条件和岗位范围由公司制度决定，变化很快。这个信息只用于理解推荐人的动机：

- 推荐人可能愿意帮忙，因为内推对他也有收益。
- 推荐人也会评估候选人匹配度，因为低质量推荐会影响他的信誉。
- 不要主动要求“奖金分成”，除非对方先提出并且双方关系足够熟。
- 不要付费购买所谓“保面试/保录用”的内推服务。

公开社区里常能看到“某公司某职级内推奖金”的经验贴或截图，但这类信息强时效、强部门差异，不能写进固定规则。需要使用时，只能作为当年当岗位的参考线索，并以目标公司内推系统或推荐人当前说明为准。

常见发放节奏也以公司制度为准，可能是入职后发放、转正后发放、分阶段发放，或只对特定岗位/职级有效。候选人不需要主动讨论奖金，除非推荐人主动说明。

## 风险与边界

- 不把简历群发给大量陌生人。
- 不把身份证、薪资流水等敏感材料发给非官方推荐人。
- 不轻信“付费内推包过简历”。
- 内推不能保证录用，只是提高被看到的概率。

## 来源与更新线索

- 内推奖金、流程和可见状态由公司内部制度决定，变化很快；使用时以目标公司当前内推系统为准。
